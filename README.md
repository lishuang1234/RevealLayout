# RevealLayout


##RevealLayout 水波纹点击效果


 实现思路:
 
1.继承LinearLayout，获取点击View

2.在绘制子元素完成时，在点击View之上绘制半径逐渐扩大的圆形

3.拦截抬起手势，延迟一段时间进行程序点击调用
